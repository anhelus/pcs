# YOLO

YOLO è un popolare insieme di modelli per la object detrection.

YOLO venne sviluppato da Joseph Redmon, Ali Farhadi e Santosh Divvala per ottenre elevata accuracy nella object detection con velocità in termpo reale. La famiglia di modelli appartiene ai modelli di object detection single-stage che elaborano un'intera immagine in un singolo forward pass di una convolutional neural network (CNN).

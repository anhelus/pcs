Cos'è una rete neurale ricorrente
Una rete neurale ricorrente (RNN) è un tipo speciale di rete neurale artificiale adattato per lavorare con dati di serie temporali o dati che coinvolgono sequenze. Le reti neurali feedforward ordinarie sono progettate solo per punti dati che sono indipendenti l'uno dall'altro. Tuttavia, se abbiamo dati in una sequenza in cui un punto dati dipende dal punto dati precedente, è necessario modificare la rete neurale per incorporare le dipendenze tra questi punti dati. Le RNN hanno il concetto di "memoria" che aiuta a memorizzare gli stati o le informazioni degli input precedenti per generare l'output successivo della sequenza.

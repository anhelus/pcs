# E1 - Introduzione a Python

## E1.1

Creiamo una stringa che assuma valore *PCS*.

### S1.1 - Soluzione

```py
>>> s = 'PCS'
```

## E1.2

Valutiamo la lunghezza della stringa creata al punto 1.

### S1.2 - Soluzione

```py
>>> len(s)
```

## E1.3

Proviamo a creare una lista a partire dalla stringa creata al punto 1.

### S1.3 - Soluzione

```py
>>> l = ['pcs']
>>> l = ['p', 'c', 's']
```
